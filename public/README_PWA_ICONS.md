# PWA Icons

This directory should contain the following icon files for the Progressive Web App:

- `icon-192.png` - 192x192 pixels icon for Android and general use
- `icon-512.png` - 512x512 pixels icon for Android and general use

## Icon Requirements

- **Format**: PNG
- **Sizes**: 192x192 and 512x512 pixels
- **Purpose**: Should be "maskable" icons that work well on various backgrounds
- **Design**: Should represent the Project Nidus brand/logo

## Creating Icons

You can create these icons using:
1. Design tools (Figma, Adobe Illustrator, etc.)
2. Online icon generators
3. Image editing software

The icons should be square and centered, with appropriate padding to ensure they look good on various device home screens.

## Current Status

⚠️ **Placeholder files are currently in place.** Replace `icon-192.png` and `icon-512.png` with actual icon images before deploying to production.
