## Introduction

Welcome to Project Nidus! This guide will help you get started with using the system as a team member.

**Your Role:**
- Complete assigned tasks
- Track time spent on work
- Collaborate with team members
- Stay updated on project progress
- Update your profile and preferences

---

## Getting Started

### Logging In

1. Visit the Project Nidus login page
2. Enter your email and password
3. If MFA is enabled, complete multi-factor authentication
4. Access your dashboard

### Dashboard Overview

Your dashboard shows:
- **My Tasks**: Tasks assigned to you
- **Recent Activity**: Latest updates from your projects
- **Upcoming Deadlines**: Tasks due soon
- **Team Updates**: Messages from team members

---

## Task Management

### Viewing Your Tasks

Navigate to **My Tasks** to see:
- All tasks assigned to you
- Task status (To Do, In Progress, Done)
- Due dates
- Priority levels

### Updating Task Status

1. Open a task
2. Change status using the status dropdown:
   - **To Do**: Task not started
   - **In Progress**: Currently working on it
   - **Blocked**: Cannot proceed (add reason)
   - **Done**: Task completed
3. Add comments or notes
4. Save changes

### Task Details

Each task includes:
- Description and requirements
- Attachments and files
- Comments and discussions
- Time tracking
- Dependencies

---

## Time Tracking

### Logging Time

1. Open a task
2. Click **Log Time**
3. Enter:
   - Date
   - Hours worked
   - Description of work
4. Save time entry

### Viewing Time Reports

Navigate to **Time Tracking** to see:
- Total hours logged
- Hours by project
- Hours by task
- Weekly/monthly summaries

## Collaboration Features

### Comments and Discussions

- Add comments to tasks
- @mention team members
- Reply to comments
- Attach files

### Activity Feed

View activity feed to see:
- Task updates
- Team member actions
- Project announcements
- System notifications

### File Sharing

- Upload files to tasks
- Share documents
- Access shared files
- Download attachments

## Notifications

### Notification Types

You'll receive notifications for:
- New task assignments
- Due date reminders
- Comments and mentions
- Status changes
- Project updates

### Managing Notifications

1. Click your profile icon
2. Go to **Notification Preferences**
3. Configure:
   - Notification channels (email, in-app, push)
   - Notification types
   - Frequency

---

## Profile Management

### Updating Your Profile

1. Click your profile icon
2. Select **Profile**
3. Update:
   - Name and contact information
   - Profile picture
   - Job title
   - Skills and expertise
   - Bio

### Security Settings

Manage security settings:
- Change password
- Enable MFA (Multi-Factor Authentication)
- Manage devices
- View login history

### Privacy Settings

Configure privacy preferences:
- Visibility of your profile
- Data sharing preferences
- Notification preferences

## Tips and Best Practices

### Staying Organized

- Review your tasks daily
- Update task status regularly
- Log time as you work
- Check notifications frequently

### Effective Communication

- Use @mentions when addressing team members
- Be specific in comments
- Ask questions early
- Share updates proactively

### Time Management

- Log time accurately
- Track time daily
- Review time reports weekly
- Identify time-saving opportunities
