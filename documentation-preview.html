Getting Started with PM Platform PM Platform

Last updated: January 2025 • Estimated read time: 10 minutes

Introduction

Welcome to Project Nidus PM Platform! This guide will help you get started with managing your projects efficiently.

Your First Project

To create your first project:

  1. Navigate to Projects from the main menu
  2. Click Create New Project
  3. Enter project details:
    • Project name
    • Description
    • Start date
    • Methodology (Structured PM, Scrum, or Kanban)
  4. Click Create

Key Features

Feature Description
Task Management Create, assign, and track tasks with dependencies
Timeline Planning Visualize project timelines with Gantt charts
Risk Management Identify and manage project risks proactively

Tip: Start with a simple project to familiarize yourself with the platform before managing complex projects.
