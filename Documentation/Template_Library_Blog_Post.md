# Template Library & Tailoring: Ship Consistent Artefacts Across Projects

We have shipped a unified **Template Library** so PMO can maintain one source of truth for registers, plans, and reports, while each project (or simulator run) keeps an independent tailored copy with full version history.

**Highlights**

- Master templates never overwritten by project users; copies are explicit.
- Automatic snapshots on save for masters and for project copies.
- Notifications when a published master template changes, so teams can merge updates intentionally.

See `Documentation/Template_Library_User_Guide.md` for routes and migration order.
