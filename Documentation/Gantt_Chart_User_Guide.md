# Gantt Chart User Guide

## Overview

The Gantt Chart is a powerful visual project planning tool that displays tasks, dependencies, and timelines in a horizontal bar chart format. It provides a comprehensive view of your project schedule, helping you understand task relationships, identify critical paths, and manage project timelines effectively.

## Accessing the Gantt Chart

1. Navigate to your project from the **Projects** page
2. Click on **Gantt Chart** in the project navigation menu
3. The Gantt Chart will load with all project tasks

## Key Features

### 1. Task Display

- **Task Bars**: Each task is represented by a horizontal bar
- **Task Duration**: Bar length represents the task duration
- **Task Status**: Color coding indicates task status:
  - Blue: Not started
  - Yellow: In progress
  - Green: Completed
  - Red: Overdue or blocked

### 2. Task Dependencies

- **Dependency Lines**: Arrows connect dependent tasks
- **Predecessor/Successor**: Tasks can have multiple dependencies
- **Critical Path**: Highlighted path showing tasks that affect project completion

### 3. Timeline Navigation

- **Zoom Controls**: Use zoom in/out to adjust time scale
- **Scroll**: Drag or use scroll bars to navigate through time
- **Today Indicator**: Red vertical line shows current date
- **Date Range**: Adjust visible date range using the date picker

## Working with Tasks

### Viewing Task Details

1. **Hover** over a task bar to see quick information:
   - Task name
   - Start and end dates
   - Duration
   - Assigned team member
   - Progress percentage

2. **Click** on a task bar to:
   - Open task detail view
   - Edit task information
   - View dependencies
   - Add comments or attachments

### Creating Tasks

1. Click the **+ Add Task** button
2. Fill in task details:
   - Task name (required)
   - Description
   - Start date
   - Duration (days or hours)
   - Assigned team member
   - Priority level
3. Click **Save** to add the task to the Gantt Chart

### Editing Tasks

1. Click on a task bar or use the task list on the left
2. Click **Edit** button
3. Modify task details:
   - Update dates
   - Change duration
   - Reassign team member
   - Update progress
   - Modify dependencies
4. Click **Save** to apply changes

### Managing Dependencies

1. **Add Dependency**:
   - Click on a task
   - Go to **Dependencies** tab
   - Click **Add Dependency**
   - Select predecessor task
   - Choose dependency type (Finish-to-Start, Start-to-Start, etc.)
   - Click **Save**

2. **View Dependencies**:
   - Dependencies appear as arrows on the Gantt Chart
   - Hover over arrows to see dependency details
   - Click on dependency line to edit or remove

3. **Dependency Types**:
   - **Finish-to-Start (FS)**: Task B starts after Task A finishes
   - **Start-to-Start (SS)**: Task B starts when Task A starts
   - **Finish-to-Finish (FF)**: Task B finishes when Task A finishes
   - **Start-to-Finish (SF)**: Task B finishes when Task A starts

## Timeline Management

### Adjusting Dates

1. **Drag Task Bars**: Click and drag task bars to change start dates
2. **Resize Bars**: Drag the ends of bars to change duration
3. **Bulk Update**: Select multiple tasks and update dates together

### Zoom Levels

- **Day View**: See daily breakdown
- **Week View**: See weekly overview
- **Month View**: See monthly summary
- **Quarter View**: See quarterly planning

### Navigation

- **Scroll Left/Right**: Use horizontal scroll bar or arrow keys
- **Scroll Up/Down**: Use vertical scroll bar to see all tasks
- **Jump to Today**: Click "Today" button to center on current date
- **Jump to Date**: Use date picker to navigate to specific date

## Filtering and Sorting

### Filter Tasks

1. Click **Filter** button
2. Select filter criteria:
   - Status (Not Started, In Progress, Completed)
   - Assigned team member
   - Priority level
   - Date range
3. Click **Apply** to filter the view

### Sort Tasks

1. Click column headers in the task list to sort:
   - Task name
   - Start date
   - End date
   - Duration
   - Assigned to
   - Progress

## Critical Path Analysis

The critical path shows tasks that directly impact project completion:

1. **View Critical Path**: Click **Show Critical Path** toggle
2. **Critical Tasks**: Highlighted in red or bold
3. **Path Analysis**: See which tasks can delay the project
4. **Optimization**: Identify tasks to prioritize or accelerate

## Export and Sharing

### Export Options

1. Click **Export** button
2. Choose export format:
   - **PDF**: For printing and sharing
   - **PNG/Image**: For presentations
   - **CSV**: For data analysis
   - **Microsoft Project**: For import into MS Project

### Printing

1. Click **Print** button
2. Adjust print settings:
   - Page orientation
   - Date range
   - Task filters
3. Preview and print

## Best Practices

### Planning

1. **Break Down Work**: Create detailed tasks (1-5 days each)
2. **Set Realistic Durations**: Base estimates on historical data
3. **Identify Dependencies**: Map all task relationships early
4. **Add Buffer Time**: Include contingency for risks

### Monitoring

1. **Regular Updates**: Update progress weekly or bi-weekly
2. **Track Delays**: Mark tasks that are behind schedule
3. **Review Dependencies**: Check if dependency changes affect timeline
4. **Update Estimates**: Adjust durations based on actual progress

### Collaboration

1. **Assign Clearly**: Ensure each task has an owner
2. **Communicate Changes**: Notify team of schedule changes
3. **Use Comments**: Add context to tasks via comments
4. **Share Views**: Export and share Gantt views with stakeholders

## Troubleshooting

### Common Issues

**Task not appearing on Gantt Chart**
- Check if task has valid start date
- Verify task is not filtered out
- Ensure task is assigned to current project

**Dependencies not showing**
- Confirm dependency is saved
- Check if both tasks are visible
- Verify dependency type is set correctly

**Timeline not updating**
- Refresh the page
- Check if dates are in valid format
- Verify task dates are within visible range

**Performance issues with large projects**
- Use filters to reduce visible tasks
- Adjust zoom level to show fewer details
- Consider breaking project into phases

## Keyboard Shortcuts

- **Home**: Jump to project start
- **End**: Jump to project end
- **T**: Jump to today
- **F**: Toggle filters
- **E**: Export chart
