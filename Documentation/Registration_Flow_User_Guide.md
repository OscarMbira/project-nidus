**Version:** 1.0

## Overview

This guide explains the new registration flow for the Platform. The flow has been redesigned to require organisation setup before project creation, ensuring better account management and trial control.

---

## Registration Flow Steps

### Step 1: Email Verification

1. **Sign Up**: Create an account with your email address
2. **Email Confirmation**: Check your inbox for the verification email
3. **Click Verification Link**: Verify your email address
4. **Automatic Redirect**: You'll be redirected to organisation setup

### Step 2: Organisation Setup (MANDATORY)

**Purpose**: Create your organisation (account) before accessing the platform.

1. **Fill Organisation Details**:
   - Organisation Name (required)
   - Organisation Type: Individual, Freelancer, Business, or Company
   - Company Legal Name (if applicable)
   - Country
   - Phone Number (optional)
   - Industry
   - Organisation Size (optional)

2. **Submit**: Click "Create Organisation"

3. **Verification Email Sent**: You'll receive an email with a verification link

### Step 3: Organisation Verification

1. **Check Email**: Look for the organisation verification email
2. **Click Verification Link**: Verify your organisation
3. **Automatic Redirect**: You'll be redirected to project type selection

**Note**: You cannot create projects until your organisation is verified.

### Step 4: Project Type Selection

Choose how you want to start:

#### Option A: Free Trial (Recommended for New Users)

- **Duration**: 14 days
- **Features**: Limited to basic project management
- **Member Limit**: 5 team members
- **Upgrade**: Can upgrade to paid at any time

**Eligibility**: One free trial per organisation

#### Option B: Paid Subscription

- **Immediate Access**: Full features from day one
- **Choose Plan**: Select from Starter, Professional, Enterprise, or Lifetime
- **Payment**: Complete payment via Paynow
- **No Limits**: Full access to all features

### Step 5: Project Setup

#### For Trial Projects:

1. **Enter Project Details**:
   - Project Name
   - Project Type
   - Description
   - Start Date
   - End Date
   - Enable Platform Module
   - Enable Simulator Module

2. **Create Trial Project**: Click "Create Trial Project"

3. **Trial Dashboard**: Access your trial project dashboard

#### For Paid Projects:

1. **Select Subscription Plan**:
   - Choose billing cycle (Monthly/Yearly)
   - Select plan tier
   - Choose number of team members

2. **Enter Project Details**: Same as trial projects

3. **Complete Payment**: Redirected to Paynow for payment

4. **Project Created**: Access your paid project after successful payment

## Trial Management

### Trial Dashboard

The trial dashboard shows:
- **Days Remaining**: Countdown to trial expiry
- **Upgrade Prompts**: Prominent upgrade buttons
- **Limited Features**: Some features are locked
- **Member Count**: Current team size vs. limit

### Trial Expiry Warnings

You'll receive email reminders:
- **3 Days Before**: First warning email
- **1 Day Before**: Final warning email
- **On Expiry**: Project is locked, upgrade required

### Upgrading from Trial

1. **Click "Upgrade"**: From trial dashboard or expiry modal
2. **Select Plan**: Choose your subscription plan
3. **Complete Payment**: Via Paynow
4. **Automatic Upgrade**: Project unlocks immediately
5. **Data Preserved**: All trial data is retained

## Organisation Management

### One Email = One Organisation

- Each email address can only create **one organisation**
- This prevents trial abuse
- If you need multiple organisations, use different email addresses

### Organisation Verification

- **Required**: Must verify organisation before creating projects
- **Email Verification**: Sent automatically after organisation creation
- **Resend**: Can request new verification email if expired
- **Support**: Contact support for manual verification if needed

## Common Issues & Solutions

### Issue: "Organisation already exists"

**Solution**: You can only have one organisation per email. If you need a new organisation, use a different email address.

### Issue: "Verification email not received"

**Solutions**:
1. Check spam/junk folder
2. Wait a few minutes (email delivery can be delayed)
3. Request a new verification email
4. Contact support for manual verification

### Issue: "Trial not eligible"

**Solution**: Your organisation has already used its free trial. Create a paid project instead.

### Issue: "Payment failed"

**Solutions**:
1. Check your payment method
2. Ensure sufficient funds
3. Try again after a few minutes
4. Contact support if issue persists

### Issue: "Project locked after trial expiry"

**Solution**: Upgrade to a paid subscription to unlock your project. All data is preserved.
